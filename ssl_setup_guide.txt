SSL SETUP COMMANDS - Run these one by one:

1. First, update Nginx config to use HTTP temporarily:
   sudo sed -i 's/listen 443 ssl http2;/listen 80;/' /etc/nginx/sites-available/sovrenai.app
   sudo sed -i '/ssl_certificate/d' /etc/nginx/sites-available/sovrenai.app
   sudo nginx -t && sudo systemctl reload nginx

2. Get SSL certificate:
   sudo certbot --nginx -d sovrenai.app -d www.sovrenai.app

   When prompted:
   - Enter email: admin@sovrenai.app
   - Agree to terms: A (agree)
   - Share email: N (no)
   - Redirect HTTP to HTTPS: 2 (redirect)

3. Verify SSL is working:
   curl https://sovrenai.app

That's it! Your site is now secure with HTTPS.
